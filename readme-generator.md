
# README Generator

## Description
This generator will help you create a README for your projects. It can help you stay organized in each of your projects by asking important questions regarding your project.

## Installation
To install this application you clone the SSH Key from the main hub page of this repository and you paste that clone link into your Terminal.
    
## Usage
To use this application you run the index.js file with 'node index.js'. After, you will answer questions based on your project.
